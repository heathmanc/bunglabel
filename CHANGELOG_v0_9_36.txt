BungVision Label Studio v0.9.36 - Canvas Performance Cleanup

Built from v0.9.35 force-review + recipe-index performance baseline.

Canvas performance changes:
- Cached the scaled display pixmap for the current image/zoom/viewport size.
- Panning now reuses the cached pixmap instead of repeatedly smooth-scaling the full 5 MP source image.
- During active pan, the canvas draws a lighter overlay: OBB outlines only, without handle numbers/text tags/model-test labels. Full detail returns when panning stops.
- OBB handle dragging no longer emits boxes_changed on every mouse-move. This prevents autosave, dataset-summary refresh, and side-panel status updates from firing continuously while a point is being dragged. The change is emitted once on mouse release.

Preserved:
- OBB battery + OBB bung workflow.
- Basler/Pylon open/capture fixes.
- Manual/auto exposure controls.
- Strict reviewed-only export/training.
- Force Review Current for intentional quantity mismatches.
- v0.9.35 recipe-index/status-cache performance cleanup.
