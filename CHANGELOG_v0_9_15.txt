BungVision Label Studio v0.9.15
================================

Live Capture tab cleanup
------------------------
- Compact camera/backend selector area so fields no longer collide or overwrite each other.
- Shortened the Basler/Pylon help text and only show it when Basler/Pylon is selected.
- Kept the Source field editable for Basler/Pylon so a serial/model filter can still be entered.
- Combined Width/Height/FPS into one compact Format row.
- Combined live preview options into one compact Options row.
- Moved Test/Open/Stop/Capture buttons into a single horizontal control row.
- Shortened button names to reduce layout crowding.

Still based on v0.9.14 behavior
-------------------------------
- Box detection workflow remains the default.
- Custom classes remain available.
- JSON status indicators remain in the captured image list.
- Clear Boxes still only clears the on-screen boxes and does not delete/overwrite saved JSON unless labels are explicitly saved.
