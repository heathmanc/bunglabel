BungVision Label Studio v0.9.20 - Class Zone Layouts

Baseline: v0.9.19 capture field polish.

Changes:
- Added battery class layout metadata for BungVision Python HMI v0.9.15 zone validation.
- Custom Classes panel now lets new classes be assigned as:
  - Custom / no zones
  - Battery generic fallback
  - Battery 6-in-row zones
  - Battery 2x3 zones
- Existing class configs are upgraded in memory with role and zone_layout fields.
- Canvas now draws visual zone guides inside battery boxes when the selected battery class is 6-in-row or 2x3.
- Added export mode: "Use annotation class names / zone classes".
- Set the new export mode as the default so battery_6row and battery_2x3 classes export unchanged for BungVision v0.9.15.
- Preserved standard YOLO detect-box export. No OBB or segmentation workflow was reintroduced.
- Preserved v0.9.19 capture field polish, Windows/PySide6 direction, manual 2592x1944 resolution defaults, JSON sidecar indicator behavior, custom classes, and Clear Boxes safety behavior.

Notes:
- Zone guides are visual label-quality helpers only. They do not create extra labels.
- Draw one battery box using the correct battery layout class, then draw normal bung boxes.
