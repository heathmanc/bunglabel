BungVision Label Studio v0.9.28 - OBB Review Status

Built from v0.9.27 OBB UI polish.

Changes:
- Added a review-status workflow for images imported from BungVision.
- Any image with a sidecar JSON containing labels but no reviewed marker now appears as REVIEW in the captured-image list.
- Added Show only needs review filter in the Live Capture tab.
- Added Find Unreviewed button and Ctrl+U menu shortcut.
- Added Mark Current Reviewed button and Ctrl+Shift+R menu shortcut.
- Save and Save + Next now write a reviewed marker into the label JSON.
- Autosave preserves any existing review metadata but does not automatically mark imported images reviewed.
- Dataset summary now includes a needs review count.

Preserved:
- OBB battery + OBB bung labeling workflow.
- Basler/Pylon open/capture fixes.
- Manual/auto exposure controls.
- Left/right panel UI polish from v0.9.27.
- No bung-zone concept.
