BungVision Label Studio v0.9.35 - Performance Cleanup + Force Review Baseline

Built from the working v0.9.34 force-review baseline.

Changes:
- Added a cached recipe image index for captured/imported image paths.
- Added per-image status cache keyed by image mtime, JSON sidecar mtime, and expected bung count.
- File-list refresh no longer reparses every JSON file after each edit/save/review operation.
- Dataset summary now uses the cached status table instead of loading every sidecar repeatedly.
- Next/Previous, Find Problem, and Find Unreviewed now use the cached recipe image list.
- Save, autosave, Mark Reviewed, Force Review, Delete JSON, Capture, and Delete Image invalidate only affected cache entries.
- Added View > Refresh recipe index and Ctrl+F5 to force a full rescan when files are modified outside the app.

Preserved:
- Force Review Current for intentional quantity mismatches.
- Reviewed-only export/training behavior.
- OBB battery + OBB bung labeling/export workflow.
- Basler/Pylon open and exposure controls.
- Launcher path fixes from v0.9.33.
